# Asteroids

Welcome to the best game of the year : **Asteroids** !

## Prerequisites

JDK 11 or higher

## Installation and usage

```sh
git clone https://github.com/comarquet/Asteroids.git
cd Asteroids
./gradlew run
```
For Windows, replace `./gradlew` by `gradlew.bat`
